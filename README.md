# Aerospace Simulation Projects

This repository contains Python-based aerospace engineering simulations developed during my learning journey in Aerospace Engineering.

## Projects

### Turbojet Engine Performance Simulator
Python program that analyzes thrust and thermal efficiency of a turbojet engine.

### NACA Airfoil Aerodynamic Analyzer
Program that generates NACA 4-digit airfoil geometry and analyzes lift characteristics.

### Aircraft Performance Analyzer
Simulation tool that calculates stall speed, drag curves, lift-to-drag ratio, and glide performance.
